# NGO Connect Web Application

## Overview

NGO Connect is a full-stack web application that connects donors with non-governmental organizations. The platform allows users to discover NGOs, explore campaigns, and support meaningful causes.

This project is developed using the MERN stack with a focus on clean UI/UX and scalability.

---

## Features

- User authentication (Login and Signup UI)
- NGO listing with filters (location, cause)
- NGO profile pages
- Campaign listing interface
- Responsive design based on Figma UI

---

## Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- React Router

### Backend
- Node.js
- Express.js

### Database
- MongoDB (planned / in progress)
